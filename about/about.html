CRAYZEE4CROCHET

About this INSANE pageeeeee!

This project is a throwback to the 2000s internet with some modern features. But also a way to read about crochet in the 2000s. While researching, I realized crochet blogs either didn't make it out of the 2000s, or it just didn't really exist. So I decided to make a modern website about 2000s crochet in a way that I think a website may look with the Y2K aesthetic. Explore each page by clicking and reading through.

Project #1 for Interaction Studio by Joanna Pak.
